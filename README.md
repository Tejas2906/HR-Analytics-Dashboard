# HR Analytics Dashboard | Power BI

An interactive **HR Analytics Dashboard** created using **Power BI** to analyze employee attrition patterns and workforce metrics.  
This project was built as part of my **learning and hands-on practice in Power BI**, focusing on data modeling, DAX, and dashboard design.

---

## Project Purpose

The goal of this project is to:
- Analyze **employee attrition**
- Identify key factors influencing employee turnover
- Understand workforce distribution by **age, salary, education, and job role**
- Practice building clean, insight-driven Power BI dashboards

## Key KPIs

- **Total Employees:** 1,470  
- **Employees Left (Attrition):** 237

- **Average Age:** 37 years  
- **Average Salary:** 6.5K
